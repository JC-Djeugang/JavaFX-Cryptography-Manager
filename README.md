# JavaFX Cryptography Manager

A JavaFX desktop application that demonstrates basic cryptography concepts through an interactive graphical user interface. This project was developed as part of a Java programming course and showcases object-oriented programming, GUI development, and software testing.

## Features

- Encrypt text
- Decrypt text
- JavaFX graphical user interface
- User-friendly application layout
- Object-Oriented Design
- Unit testing included

## Technologies Used

- Java
- JavaFX
- Eclipse IDE
- Git
- GitHub
